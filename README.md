# Student Registration Database

## Overview

This project focuses on the design and implementation of a student registration database. It covers the complete database development process, including relational database design, SQL implementation, PL/SQL programming, stored procedures, and cursor operations.

## Features

- Database Design
- Entity Relationship (ER) Modeling
- SQL Table Creation
- Primary and Foreign Key Constraints
- Stored Procedures
- Data Insertion
- PL/SQL Programming
- Cursor Operations
- SQL Queries

## Database Entities

- Student
- Major
- Subject
- Academic Advisor
- Professor
- Registration

## Technologies Used

- Oracle Database
- SQL
- PL/SQL

## Purpose

This project demonstrates the implementation of relational database concepts using SQL and PL/SQL, including database design, stored procedures, cursor operations, and data retrieval.
